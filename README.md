Program ini merupakan Queen Game Solver yang memberikan penyelesaian terhadap penempatan Queen di dalam sebuah papan n x n dengan area warna berjumlah n juga. Queen berjumlah n harus ditempatkan di papan dengan syarat tidak boleh berada di baris, kolom, dan daerah yang sama.

Program ini menggunakan algoritma brute force dengan backtracking. Algoritma brute force yang diimplementasikan adalah bentuk murni tanpa heuristik.

## Requirement
- Compiler C: GCC

## Struktur Input
Input berada di file berekstensi .txt yang berisi kumpulan huruf dalam bentuk matriks. Huruf-huruf yang ada melambangkan daerah. 

## Cara Kompilasi
Jalankan perintah di folder root : 

```bash
gcc src/main.c -o bin/main
```

## Cara Menjalankan 
Dari Windows : 
```bash
bin\main.exe
```

Dari Linux : 
```bash
./bin/main
```
